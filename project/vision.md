# Vision

This project exists so that casual friends can create groups and bet on World Cup matches better than today.

## Target users

Casual friend groups — students betting a crate of beer, colleagues running an office pool. Low stakes, high fun.

## Top jobs to be done

1. Create a group and invite friends
2. Place bets on matches
3. Check the leaderboard regularly to see who's winning

## Non-goals

- Not a social network
- Not a mobile native app
- Not a professional/solo betting app

## Constraints

- Deadline: ASAP — World Cup final is 2026-07-19
- Budget: None yet
- Stack: NestJS (TS) + PostgreSQL + React, deployed on Railway (strong user preference; architect to respect unless compelling reason to deviate — ADR-001)
- Code repos: backend (`https://github.com/DrRotstein/world-cup-bets-backend`) + frontend (`https://github.com/DrRotstein/world-cup-bets-frontend`)
