## Context

The leaderboard is why people come back. After each match, points are calculated and the ranking updates. This is the dopamine loop.

## Scoring rules (D-003)

- **4 pts** — exact score (predicted 2-1, result 2-1)
- **2 pts** — correct goal difference (predicted 3-1, result 2-0 — both have diff of 2)
- **1 pt** — correct outcome only (predicted home win, got home win, but score/diff wrong)
- **0 pts** — wrong outcome

## Scope

- Point calculation triggered after match results sync
- Leaderboard table (or computed view): userId, groupId, totalPoints, exactCount, diffCount, outcomeCount
- GET /groups/:groupId/leaderboard — ranked list
- GET /groups/:groupId/leaderboard/:userId — per-match breakdown
- Frontend: leaderboard page with rank, name, avatar, points, bet history per match
- Tiebreaker: most exact predictions first, then alphabetical by name

## Non-goals

- No global leaderboard across all groups
- No historical leaderboard snapshots (just current state)
- No badges or achievements
