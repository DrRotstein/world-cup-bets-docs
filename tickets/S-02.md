## Context

Groups are the social unit. Friends create a group, share the link in their WhatsApp/Signal/whatever chat, and everyone joins. No friend requests, no approval flow — just a link.

## Scope

- Create group: name (required), optional description
- Generate unique invite code / shareable URL
- Join group via invite link (redirects to auth if not logged in, then auto-joins)
- List my groups (dashboard)
- View group members
- Group table: id, name, description, inviteCode, createdBy, createdAt
- Membership table: userId, groupId, joinedAt

## Non-goals

- No group admin roles (creator has no special powers for MVP)
- No leave/kick functionality for MVP
- No group size limit
- No group deletion
